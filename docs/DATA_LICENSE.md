# Data License Policy

The Apache-2.0 repository license covers project code and original documentation. It does not automatically grant redistribution rights for third-party facts, documents, databases, trademarks, or copyrighted source material.

Every distributed dataset must record source and license status. Public exports may contain facts derived from permitted public sources, original classifications, and project-generated analysis when legally reusable. They must not reproduce restricted databases or substantial copyrighted text.

Commercial or restricted sources may be referenced as evidence only when their terms permit it. Keep privately licensed material outside the repository and public generated artifacts.

Before a formal public dataset release, define a dataset-specific license and complete a source-by-source redistribution review.
