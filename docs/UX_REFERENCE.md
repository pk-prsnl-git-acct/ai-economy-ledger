# UX Reference — Ledger Dark

Ledger Dark is a calm, dark-first financial intelligence interface. It should feel analytical and trustworthy rather than promotional, neon, or speculative.

## Visual principles

- clear hierarchy and compact financial cards
- accessible contrast and responsive layouts
- confidence, freshness, sample, and estimate states visible near values
- source and methodology access from every headline metric
- dense tables with readable alignment and units
- restrained cyan, green, amber, red, and violet semantic accents

## Core components

`AppShell`, `TopNav`, `HeroSection`, `KpiCard`, `ConfidenceBadge`, `FreshnessBadge`, `SourceLink`, `DataQualityPanel`, `FinancialChartCard`, `DataTable`, and `SampleDataWarning`.

The handoff mockups are references, not production specifications. All displayed numbers are placeholders. Accessibility and design review accompany the implementation PR.
